Este projeto é um formulário multi-step desenvolvido em React, com o objetivo de dividir o preenchimento de dados em etapas, tornando a experiência do usuário mais intuitiva e organizada.
Cada etapa possui validações próprias e o usuário pode navegar entre os passos sem perder as informações já preenchidas.
